# Laravel + VueJs
Basic Admin For Laravel with VueJs.

## Version

Laravel Framework 5.7.6

This is a Multi-Purpose Laravel + Vue Application. 

## Installation

It's just like any other Laravel project. Basically here is how you use it for yourself. (it's not completed yet) 

* Clone the repo ` git clone https://github.com/ozairasmat/LaravelVue_adminpanel.git `
* `cd ` to project folder. 
* Run ` composer install `
* Save as the `.env.example` to `.env` and set your database information 
* Run ` php artisan key:generate` to generate the app key
* Run ` npm install ` 
* Run ` php artisan migrate ` 
* Registration (register a user with name admin)
* In database user table you should change userType from 'user' to 'admin' 
* Install Passport `php artisan passport:install`
* Done !!! Enjoy Customizing and building awesome app
